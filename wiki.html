starfrost CSS 框架

现代化、轻量级、高度可定制的CSS框架

v3.0.0 响应式 40+组件 主题切换

框架介绍

starfrost 是一个现代化的 CSS 框架，专为快速开发现代网站而设计。它提供了40+精心设计的组件、实用的工具类，以及强大的主题定制功能。

主要特性

  • 响应式设计，适配各种设备
  • 深色/浅色主题一键切换
  • 40+ 精心设计的UI组件
  • 基于CSS变量的主题定制
  • 轻量级，无依赖，仅需引入CSS与JS文件
  • 文档和示例

主题切换

starfrost 支持一键切换深色/浅色主题，并在切换时提供视觉反馈。

使用指南

安装方法

下载文件并自行引入：

https://github.com/StarsX-Studio/StarFrost/blob/main/starfrost.min.css

https://github.com/StarsX-Studio/StarFrost/blob/main/starfrost.min.js

<link rel="stylesheet" 
      href="starfrost.min.css">

基本结构

<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <title>starfrost示例</title>
  <link rel="stylesheet" href="starfrost.min.css">
</head>
<body>
  <nav class="sf-navbar">
    <!-- 导航内容 -->
  </nav>
  
  <div class="sf-container">
    <!-- 页面内容 -->
  </div>
</body>
</html>
